Option Reference

Here is a rundown of all the options you can use with Facebook Traffic Pop. All of these options can be called using the standard method for calling jQuery options in a function {}.

  • title - This is the title text that will appear in the title bar of your pop up. For best results keep it short and sweet.
  • message - The call to action message or any HTML markup you want to be in the message space. (Must be all one line if HTML.)
  • url - The url that will be 'Liked' when the user clicks like. You can also set this option to 'current' to use whatever the address of the current page is as well.
  • showfaces - true or false to show user icons when the user clicks like.
  • timeout - The number of seconds the timer will run for in the bottom right of the pop up before closing. You can also set timeout equal to 0 to disable the timer completly and require the user to like the page before continuing or press the close button.
  • delay - The delay timer can be used to keep the popup from running for x amount of seconds to allow the user to read some of the page before Facebook Traffic Pop appears.
  • closeable - true or false to display the close button in the top right corner
  • advancedClose - true or false to enable / disable the use of closing Facebook Traffic Pop with the escape key or by clicking outside the popup.
  • wait - the number of minuets you would like to wait before each pop up. For example if set to 3, the user will see the pop up every 3 minuets they are on the site. Default is 0 - no wait or pop up shows every page load.
  • lang - set the localization of the pop up. Default is 'en' for english. Language options: en, de, fr, es, it
  • opacity - Set the background opacity. Default is 25.
  • onClick - Here you can set the class to use with the onClick handle. By default the script will launch the popup when any element with the class fbtp-open is clicked. So for example you have a link with the class "custom-class" you would specify "custom-class" for this option.
