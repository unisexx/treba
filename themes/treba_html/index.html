  • สรุปแนวตัดถนนใหม่ข้ามมอเตอร์เวย์
    เวนคืนบ้านเรือนประชาชนแค่ 9 หลัง
    เมื่อเวลา 13.30 น. วันที่ 3 ก.ค. 56 ที่โรงแรมเดอะ แกรนด์โฟว์วิงส์ คอนเวนชัน สำนักการโยธา (สนย.)สัมมนารับฟังความคิดเห็นครั้งที่ 3
    ( 04 กรกฎาคม 2556 )
  • ซีคอนฯดันบ้านพรีแฟบสู้วิกฤต
    ซีคอนฯ รับมือต้นทุนวัสดุ -ค่าก่อสร้างพุ่งดึงระบบพรีแฟบใช้ชิ้นส่วนสำเร็จรูปสร้างบ้านเต็มรูปแบบ รับความต้องการของกลุ่มลูกค้ากลุ่มใหญ่ในตลาดกลาง-ล่าง แจงขึ้นราคาบ้าน 5%
    ( 04 กรกฎาคม 2556 )
  • เปิดรับสมัครสอบบุคคลเพื่อรับบัตรฯ
    สมาคมนายหน้าอสังหาริมทรัพย์ไทย เปิดรับสมัครสอบบุคคลเพื่อรับบัตรประจำตัว "นายหน้าอสังหาริมทรัพย์ไทย" ครั้งที่ 3/2556 รุ่นที่ 51 กำหนดการจัดสอบ วันเสาร์ที่ 6
    ( 04 กรกฎาคม 2556 )
  • บางพลัดค้านเวนคืนรถไฟสีน้ำเงินแนะกทม
    เจาะอุโมงค์ใต้ดินแทน
    ประชาชนชาวบางพลัดยื่นหนังสือรองผู้ว่าฯ กทม. ขอความชัดเจนเรื่องเวนคืนที่ดินสร้างรถไฟฟ้าสายสีน้ำเงิน ชี้ประชาชนเดือดร้อนหนัก
    ( 04 กรกฎาคม 2556 )
  • บ้าน-คอนโดฯแห่ผุด รังสิต หมื่นยูนิต
    ตอบโจทย์แรงงาน-นักศึกษา
    บ้าน-คอนโดฯในพื้นที่รังสิตกลายเป็นทำเลทองที่กลุ่มผู้ประกอบการอสังหาริมทรัพย์ให้ความสนใจหลังเหตุการณ์น้ำท่วมรุนแรงในช่วงปลา
    ( 04 กรกฎาคม 2556 )
  • เปิดอบรมผู้สนใจเป็นตัวแทนขายฯ
    เปิดอบรมผู้สนใจเป็นตัวแทนขายธุรกิจนายหน้าอสังหาริมทรัพย์ รุ่นที่ 3/2556 ในวันที่ 21-22-23 มิถุนายน 2556 ตั้งแต่บัดนี้เป็นต้นไป
    ( 04 กรกฎาคม 2556 )
